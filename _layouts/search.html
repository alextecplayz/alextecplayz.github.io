Information about how this works, below:

Expand to show the contents of this collapsed section

The search retrieves the website's sitemap.xml, retrieves certain XML tags including the XML loc tag and custom tags under the Vedalia namespace (custom namespace by me), such as vedalia:title, vedalia:description, vedalia:date, vedalia:content, goes through it for each URL listed in the sitemap, and displays the results. Each result has a snippet below, where the first iteration of the query is visible, while offering some context by providing 150 characters before and after the query itself.


You can, of course, check the code itself on GitHub to take a more in-depth look, if so you prefer. Additionally, you can browse the sitemap generator's code here.


Attribute (tag) Required or Optional Description
vedalia:layout optional The layout, such as 'post', 'page', read from page.layout
vedalia:type optional The type, such as 'post', 'page', read from page.type
vedalia:lang optional The language tag, such as 'en', read from page.lang
vedalia:locale optional Granular language tag that indicates the locale, such as 'en_US' or 'en_GB', read from page.locale
vedalia:title optional The title of the post or page
vedalia:description optional The description of the post or page
vedalia:date optional The date and hour of the post or page, of the format YYYY-MM-DDTHH:MM:SS+HH:MM, for example '2024-10-24T21:00:00:00+03:00'
vedalia:indicator_type optional The type of the indicator, such as 'notice'.
vedalia:indicator_class optional The class of the indicator, such as 'notice-danger', 'notice-warning' or 'notice-caution'.
vedalia:indicator_text optional The shortened text of the indicator, such as 'Warning', 'Experimental', 'Caution', 'Danger', 'Info'.
vedalia:indicator_text_expanded optional The 'on page' text of the indicator, read from page.indicator_text_onpage. This is usually more detailed in explaining the purpose or state of the present indicator.
vedalia:thumbnail_url optional The URL of the thumbnail image, relative to the website's root.
vedalia:thumbnail_alt optional Alternative (ALT) text for the thumbnail image, if available
vedalia:categories optional A list of categories. Parent of as many vedalia:category XML tags as the number of categories listed in the YAML
vedalia:category optional One category tag per category. Child of vedalia:categories XML tag
vedalia:tags optional A list of tags. Parent of as many vedalia:tag XML tags as the number of tags listed in the YAML
vedalia:tag optional One tag. Child of vedalia:tags XML tag
vedalia:content optional The (escaped) HTML content of a page or post. This is searched through and snippets are using the provided HTML markup to render accordingly.
