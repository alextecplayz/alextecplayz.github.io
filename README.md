### alextecplayz.com
My website. It has a blog and pages. It used to run on Jekyll, now it has moved to 11ty.

##### Run it locally:

Requirements:

- [nvm](https://github.com/nvm-sh/nvm)
- Node lts/iron or newer (recommended, install via nvm)

```
git clone https://github.com/alextecplayz/alextecplayz.github.io.git atp-web && cd atp-web && npm install && npx @11ty/eleventy --serve --incremental
```
