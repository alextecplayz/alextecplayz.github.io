Here's how I calculate my ratings for the games I review.

### Game quality

**1 point** - **are the game's core gameplay mechanics solid, fun, intuitive?** // Most games should get this point. Even my [Tomb Raider 2013 review]({{site.baseurl}}/posts/2025-02-10-Tomb-Raider-2013-Review.html) met these criteria, as the core gameplay loop and mechanics are great and there's not much to criticize there.

**1 point** - **is the music / soundtrack of the game good? How is the audio quality?** // *Only applies to games that have music. In outlier cases where music is not available, the general audio quality is judged instead*

**1 point** - **is the campaign / story / lore good?** // Is the story and its associated lore not convoluted, is it easy enough to follow, does it have the expected structure of a plot (introduction, story, ending), the presence of conflict, the perspective(s) used in narration, the setting, theme and characters?

**1 point** - **the quality of graphics and art style** // I'm not talking about how 'modern' a game is, what rendering technologies and techniques are used - as that would immediately disqualify many old games, or games that use older rendering techniques, and some 2D games. I'm specifically talking about the quality of the graphics, in the sense that they're not 'the default look' from a game engine, but specifically tweaked for the game's purposes, and the associated art style and art direction that goes with it.

**1 point** - **longevity and replayability** // I'm talking about post-launch support, or if there's caution to be had if the game relies on an Internet connection and servers, if there's mod support, New Game+, etc.

### Consumer views

**1 point** - **is the game accessible?** // Are there any accessibility features in the game?

**1 point** - **is the game accessible offline?** // Are there any always-online requirements to unlock or access content that should theoretically be available fully offline (e.g. campaigns, campaign DLCs, unlockable cosmetics and items, challenges)

**1 point** - **is the game's price reasonable for the value it provides?** // In most cases, games priced $60 and higher do not provide reasonable value for the customer. This includes most Call of Duty games, BATTLEFIELD, and so on. The single-player campaign of a Call of Duty game should cost roughly $20, so the rest is padded through the multiplayer 'value' of having unlockable cosmetics through grind or microtransactions, season passes, additional multiplayer content post-launch, and the occasional Zombies game mode, which is, in recent titles, of questionable quality and not really worth it.

**1 point** - **is the developer and/or publisher ethical?** // This category concerns actions from developers and/or publishers regarding microtransactions, DLCs, if they're using generative AI in a thoughtful, restrained and *genuinely useful* way, their use of game editions, price schemes, FOMO tactics, and so on.

**1 point** - **DRM and -NIX support (native or via Wine/Proton/other compatibility layers)** // Not necessarily targeting platform DRM such as Epic Games or Steam DRM, but specific DRM such as Denuvo, other third-party DRMs that are considered spyware/bloatware, DVD DRM such as SecuROM, and if the game is playable on Linux either through the use of a native Linux version or via compatibility layers such as Wine or Proton, and optionally on -BSD systems.

---

This is my site, and my reviews, so this is my rating system. A game that may be 10/10 on other sites may or may not also be 10/10 in my review using this score. A good game with no accessibility features, an always-online single-player campaign, sold for $60, with an unethical publisher using aggressive microtransaction tactics, loads of DLCs, using Denuvo and/or not being playable on Linux would only net it a 5/10 on my site.

Note that I don't use fractional scores. I don't have 8.5/10 or so on. Internally, the 1 point for each category is divided into two - below threshold and above threshold. The threshold is serviceability. For example, if the game's core mechanics are fine (not standing out from other games, but they're not bad, and I don't have any major concerns or complaints), that would be above the threshold and be awarded the point. Either way, of course I detail everything in the review I publish for each game.
